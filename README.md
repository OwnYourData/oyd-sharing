# OwnYourData Sharing

OwnYourData's application for sharing data.

## Installation

```bash
npm install
```

## Development

```bash
npm run serve # starts transpiling and the dev-server
```

## Build for production

```bash
npm run build
```

---

This project was bootstrapped using vue-cli
